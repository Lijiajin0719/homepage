# 个人学术主页设计规格书

**日期**: 2026-05-13
**用户**: 李佳津
**方向**: 深度学习医学图像处理

---

## 1. 项目概述

构建一个多页个人学术主页，展示论文发表、研究项目、竞赛获奖、专利软著、参会情况、教育背景等内容。内容来源于PPT文件（1028403105_李佳津.pptx）。

### 核心需求
- 多页应用（5个页面），顶部导航切换
- 中英文切换（UI文字可切换，论文标题/专利名称保持原文）
- 学术简约风格：白色主调 + 深蓝 `#1e3a5f` / `#2b6cb0` 强调色
- 全响应式设计（桌面端/平板/手机）
- 使用PPT中提取的32张图片

---

## 2. 技术方案

纯静态 HTML/CSS/JS，零框架依赖，无构建步骤。

### 文件结构
```
个人主页/
├── index.html              # 首页
├── publications.html       # 论文成果
├── research.html           # 研究项目
├── achievements.html       # 获奖成果
├── about.html              # 个人履历
├── css/
│   └── style.css           # 全局样式 + 响应式
├── js/
│   ├── i18n.js             # 中英文切换数据与逻辑
│   └── main.js             # 导航、交互
└── assets/
    └── images/             # PPT提取的32张图片
```

### i18n 方案
- 所有UI文字通过 `data-i18n` 属性标记
- `i18n.js` 维护中英文映射表
- 导航栏语言切换按钮切换 `document.documentElement.lang`
- 论文标题、专利名称等专业内容保持原文不翻译

## 3. 页面设计

### 3.1 首页 (`/`)
**布局**: 左右分栏

| 左侧 (约30%) | 右侧 (约70%) |
|-------------|-------------|
| 头像/姓名缩写 | 个人简介卡片 |
| 姓名、身份 | 研究方向卡片 |
| 关键数据矩阵(2×2)：论文7/专利1/软著5/国奖2 | 最新动态卡片 |

### 3.2 论文成果 (`/publications`)
**布局**: 纵向列表卡片

- 顶部：筛选标签（全部 / 已接收 / 返修中）
- 每条论文为独立卡片：
  - 左侧色条：蓝=已接收，黄=返修中
  - 论文标题（英文原文）
  - 作者列表
  - 期刊/会议名称 + 年份
  - 中科院分区 + IF标签
- 会议论文分组展示

### 3.3 研究项目 (`/research`)
**布局**: 纵向展开卡片 + 锚点导航

- 顶部：圆形标签导航（精子分析 / FPM成像 / 超声重建 / 光声成像）
- 每个研究方向一个卡片：
  - 示意图横幅（来自PPT图片）
  - 研究标题
  - 背景与挑战描述
  - 方法与创新点
  - 实验环境
  - 研究成果标签（论文、专利、软著、竞赛）

共4个研究项目：
1. 基于深度学习的实时精子质量评估 (MTCA-Net + HOPE-Net)
2. 基于小波变换和线性注意力的快速高保真FPM成像 (WM-FPM)
3. 基于深度学习的超声平面波重建 (U2-Net + DP-HAFNet)
4. 博士规划：物理先验约束的深度学习光声成像去噪

### 3.4 获奖成果 (`/achievements`)
**布局**: 2×2 分类卡片网格

- 学科竞赛卡片：国赛一等奖×2 + 国赛二等奖×1
- 国家发明专利卡片：已授权1项
- 软件著作权卡片：已登记5项
- 学术会议卡片：口头汇报1次

### 3.5 个人履历 (`/about`)
**布局**: 时间线 + 信息卡

- 教育经历：左侧时间线（本科→硕士）
- 研究方向：2×2 信息卡矩阵
- 科研基金参与情况
- 联系方式
